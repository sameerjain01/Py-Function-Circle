 Py-Function-Circle
 ====
 
# This python function receive an integer radius value to provide area of circle.

# Behaviour
expect an integer value of radius without any decimal.
* #### Valid value: 1, 3, 100 etc.
* #### Invalid values: 1.2, 1.33, 12.0, 13.0, 343.1 etc
